# Assignment 3 – Data Engineering Fellowship (Buildables)

This repository contains my **Assignment 3** from the Data Engineering Fellowship at Buildables.

## 📄 Contents
The assignment covers a wide range of data engineering concepts, including:
- Data Warehouse, Data Lake, and Data Lakehouse
- Star Schema design
- ETL vs ELT (pros, cons, tools, and use cases)
- Batch vs Streaming vs Micro-batch processing
- Medallion Architecture (Bronze, Silver, Gold layers)
- Data ingestion methods
- Data quality dimensions
- Schema evolution (forward/backward compatibility)
- File formats (CSV vs Parquet)
- Partitioning
- Orchestration tools (Airflow)
- Distributed computing (horizontal vs vertical scaling)
- Indexing and caching in databases
- Compression techniques (Snappy, Gzip, etc.)
- Data observability tools
- REST vs GraphQL APIs
- Pipelines (ingestion → storage → transformation → consumption)
- Messaging patterns (Kafka, RabbitMQ, Pub/Sub)
- Data contracts

📂 The full solutions and explanations are available in the PDF file included in this repo.

## 🚀 My Learning
This assignment helped me connect the dots between theory and practice in data engineering, from how raw data is stored, cleaned, and processed, to how it flows through modern pipelines.
